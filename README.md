# 新楓之谷ARC計算機

### 當前版本對應的遊戲版本為 TMS v215.2

- 巴哈姆特詳細介紹文章：[點我](https://forum.gamer.com.tw/C.php?bsn=7650&snA=1000541)

### 若版本號未及時更新可參考以下條件確認是否適用：

1. 秘法符文的等級上限為20等
2. 共有6種秘法符文

### 編譯方式：
- 使用Qt 5.12.0以上版本並使用MSVC 2017 32bit編譯
